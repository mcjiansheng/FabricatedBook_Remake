## 四、卡牌系统

卡牌生效说明：卡牌生效顺序按照文字描述顺序，如痛击：造成 8 点伤害，提供 2 点脆弱，造成8伤害文字描述位于2脆弱之前，则痛击打出时若目标单位未处于脆弱状态，则痛击造成的伤害先与脆弱生效，因此不会受到脆弱的增伤

### 4.2 卡牌类型说明

| 类型 | 说明 |
|:----|:-----|
| **攻击** | 直接造成伤害 |
| **技能** | 提供格挡、治疗、抽牌、临时 Buff/Debuff、一次性特殊效果等 |
| **能力** | 提供持续规则改变或长期战斗效果；当前暂无能力卡牌 |
| **状态** | 通常由战斗或敌人生成，污染牌组或限制行动；当前暂无状态卡牌 |
| **诅咒** | 通常为负面牌，进入牌组后带来持续代价；当前暂无诅咒卡牌 |
| **任务** | 用于特殊目标、剧情或路线机制；当前暂无任务卡牌 |
| **消耗** | 使用后进入消耗牌堆，不进入弃牌堆；消耗牌堆不参与本场战斗洗牌 |
| **保留** | 回合结束时仍留在手牌中，不进入弃牌堆 |
| **虚无** | 回合结束时若仍在手牌中，则进入消耗牌堆 |
| **X 费** | 打出时消耗当前所有剩余能量，消耗量记为 X |
