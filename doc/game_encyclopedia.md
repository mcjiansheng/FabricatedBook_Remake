## 一、游戏概述

### 1.1 游戏背景

玩家将扮演一名冒险者，穿越层层险境，从荒野一步步攀登至高塔，战胜最终的敌人。游戏融合了卡牌构筑、回合制战斗、随机地图探索等元素。

### 1.2 基础规则

- 玩家通过卡牌进行攻击、防御和施放技能
- 每回合自动获得 **3 点能量**，使用卡牌消耗能量
- 战斗胜利后获得金币和卡牌奖励
- 在地图节点间移动，遭遇战斗、事件、商店等
- 共 **5 层**地图，每层有独特的敌人和环境效果

### 1.3 角色系统

游戏包含 **三种职业**，每职业有专属卡牌池和特性：

| 职业 | 特性 | 生命值上限 |
|:----|:-----|:---------:|
| 🗡️ 战士 | 战斗胜利回复 6-12 点生命值 | 80 |
| 🔮 法师 |（待补充）|（待补充）|
| 🧙 女巫 |（待补充）|（待补充）|

## 三、节点系统

### 3.1 战斗节点

#### 🗡️ 作战（Fight）
常规战斗节点，胜利后获得金币和卡牌奖励。

#### ⚠️ 紧急作战（Emergency）
有一定难度增益的战斗节点，胜利后获得藏品、金币和卡牌。敌人更强，奖励更丰厚。

#### 👹 险路恶敌（Boss）
每层的最终首领战，难度极高，胜利后进入下一层。

### 3.2 事件节点

#### ❓ 不期而遇（Unexpectedly）
随机事件节点，进入后将触发未知事件（详见[事件列表](#九事件列表)）。

#### 🎁 得偿所愿（Reward）
从随机三个藏品种选择一个获得。

| 稀有度 | 出现概率 | 说明 |
|:------|:-------:|:-----|
| ⚪ 普通（价值0） | 30% | 谢谢！ |
| 🔵 稀有（价值1） | 25% | 拿走一件藏品 |
| 🟣 史诗（价值2） | 20% | 许愿…… |
| 🟡 传说（价值3） | 15% | 我需要的是这个！ |
| 🔴 神话（价值4） | 10% | 你曾在神话中所闻的物件…… |

#### 🏪 诡异行商（Shop）
商店节点。出售 **7 张卡牌 + 3 个藏品 + 3 个药水 + 1 个弃牌机会**。

**定价规则：**
- 卡牌：价值 1 → 50-70 金币 / 价值 2 → 90-110 金币
- 藏品：`价值 × 25 + (55-70 随机)` 金币
- 药水：30-70 金币（随机）
- 弃牌：初始 75 金币，每次购买 +25 金币（每商店限一次）

#### 🔀 命运抉择（Decision）
固定位置刷新，根据选择更改探索方向或获得不同效果。

**第一层命运抉择 — 迷雾：**
> 迷雾渐起，前途未知，你将何去何从？
- **前进**：突破迷雾，进入森林
- **回头**：结束游戏（获得隐藏结局「讲述中断」）

#### 🏠 安全屋（Safe House）
提供补给的休息节点。选项：

| 选项 | 效果 |
|:----|:-----|
| 💤 好好休息 | 回复 20% 生命值 |
| 💊 进行治疗 | 花费 50% 金币，恢复全部生命值 |
| 📦 获取补给 | 获得 50-200 金币（0.1% 概率获得 999 金币） |

### 4.2 卡牌类型说明

| 类型 | 说明 |
|:----|:-----|
| **攻击** | 直接造成伤害 |
| **防御** | 获得格挡值 |

| **装备** | 提供持续战斗效果 |
| **消耗** | 使用后进入消耗牌堆，不进入弃牌堆；消耗牌堆不参与本场战斗洗牌 |

| **虚无** | 回合结束时若仍在手牌中，则进入消耗牌堆 |
| **X 费** | 打出时消耗当前所有剩余能量，消耗量记为 X |

## 五、效果（Buff）系统

### 5.1 负面效果（Debuff）

| Buff | 英文 | 效果 |
|:----|:----|:-----|
| 🟡 脆弱 | Fragile | 受到伤害增加 25% |
| 🔵 易碎 | Block Reduction | 获得的格挡 -50% |
| 🟣 虚弱 | Weak | 造成的伤害减少 25% |
| 🔴 眩晕 | Dizziness | 下一回合无法行动 |
| 🟢 中毒 | Poisoning | 回合开始时依层数受到伤害，层数 -1 |
| ⚫ 凋零 | Withering | 回合开始或受特定技能引爆，每次引爆伤害增加 |

### 5.2 正面效果（Buff）

| Buff | 英文 | 效果 |
|:----|:----|:-----|
| 🛡️ 抗性 | Resistance | 受到伤害减少 25% |
| 💪 坚强 | Block Increase | 获得格挡 +50% |
| ⚡ 力量 | Strength | 造成的伤害增加 25% |
| 🛡️ 装甲 | Armor | 格挡值不再在回合结束时消失 |
| 💀 不死 | Undead | 血量降为 0 也不会死亡 |
| ⚡ 额外能量 | Extra Energy | 每回合额外获得能量 |

### 5.3 效果机制说明

- **脆弱 + 虚弱** 的组合可以显著削弱敌人
- **中毒** 是持续伤害，每回合递减
- **凋零** 可被引爆造成爆发伤害，引爆后伤害逐次增加
- **装甲** 在装备卡牌后生效，格挡不会每回合重置
- **不死** 是搏命挣扎提供的效果，3 回合后角色会直接死亡

## 十、结局一览

### ❌ 失败
> 在战斗中生命值变为 **0**，战斗失败
- 条件：任意战斗中生命值归零

### 📖 讲述中断（隐藏结局）
> 这一切都太过诡异，你决定立马离开
- 条件：在第一层命运抉择中选择「回头」

### 🏆 故事终局（普通结局）
> 你战胜了一个俗套故事中的最终 boss，它无恶不作，损人不利己，为了为恶而为恶，它或许有着悲惨的过往但不值得同情
- 条件：在高塔中战胜「魔王」

### 🌊 沉入虚妄（隐藏结局）
> 诸天神魔？以玩弄人间为乐，你？魔王？无外乎一场取悦的游戏罢了。  
> 反抗？背后的背后，请神明睁眼，告诉我一切为何发生。
- 解锁路径：
  1. 在「相遇」事件中获得「背叛」或「仇恨」
  2. 在「追猎」事件中选择「反抗」，获得「复仇者」
  3. 获得藏品「巴别塔」
  4. 在高塔中遇见隐藏 Boss「幕后黑手」并战胜
